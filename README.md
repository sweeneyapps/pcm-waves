# pcm-waves
fun project (work in process)

demo at http://sweeneyapps.com:3000/

------
## Installation Instruction
- clone this repo
- install ffmpeg (http://ffmpeg.org/download.html)
- npm install
- node app

you can change the sound file by variable filename in app.js
